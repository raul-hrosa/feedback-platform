# ADR-010 - API First

## Status

Accepted

## Contexto

O produto possui frontend web atualmente, mas poderá possuir aplicativos móveis e integrações futuras.

## Decisão

Toda regra de negócio será exposta através da API.

O frontend atuará apenas como consumidor.

## Benefícios

- Reutilização
- Escalabilidade
- Facilidade de integrações futuras
