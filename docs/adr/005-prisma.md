# ADR-005 - Utilizar Prisma ORM

## Status

Accepted

## Contexto

Necessidade de produtividade, migrações e tipagem forte.

## Decisão

Utilizar Prisma ORM.

## Alternativas Consideradas

- TypeORM
- Sequelize
- Drizzle

## Motivos

- Excelente suporte TypeScript
- Migrações simplificadas
- Boa experiência de desenvolvimento

## Consequências

- Dependência de geração de código
- Algumas limitações em consultas extremamente avançadas
