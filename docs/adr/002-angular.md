# ADR-002 - Utilizar Angular para Frontend

## Status

Accepted

## Contexto

A aplicação possui dashboards administrativos, formulários complexos e fluxo de autenticação.

## Decisão

Utilizar Angular como framework frontend.

## Alternativas Consideradas

- React
- Vue

## Motivos

- Forte tipagem com TypeScript
- Arquitetura opinionada
- Experiência prévia da equipe
- Integração natural com Nx

## Consequências

- Maior estrutura inicial
- Curva de aprendizado maior para novos desenvolvedores
